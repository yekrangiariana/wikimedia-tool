# Commons Desk

Commons Desk is a small browser tool I built for myself to speed up image work for static publishing. It helps me find Wikimedia Commons assets, prep them quickly, and generate publish-ready shortcode without switching between a bunch of tabs and tools. If you run a media site with a static workflow, it can be useful for you too.

The core flow is simple: search, review, caption, copy, and download. You can filter and sort results, inspect metadata, generate shortcodes in different layout modes, and keep everything moving from a single Finder panel.

There is an integrated editor with crop, background tools, and eraser support, plus edit undo/redo and reusable edit state. Edited images, copied captions, and downloads are tracked in Vault so previously used assets are easy to reopen and reuse.

It also supports gallery-oriented work, including saved collections, per-image caption and layout refinement, and quick action menus for common operations. Preview caching and local persistence keep repeat actions fast, so the tool stays responsive even when you are iterating on the same images.
